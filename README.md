# Wifi_debug 無接線除錯
1.開啟命令提示字元
開啟 -> 搜尋 -> cmd

2.進入到 android jdk platfrom-tool 中
cd 路徑\platform-tool

3.使用 adb 指令開啟手機網路功能
adb tcpip 5555

4.使用 adb 查看手機網路位置
adb shell ifconfig

5.使用 adb 與手機做連線
adb connect 手機 IP 位置

6.完成手機 WI-FI debug 設定
